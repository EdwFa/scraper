# Выгрузка уведомлений ЕРВК (Парсер)

Этот проект представляет собой веб-приложение на базе **Streamlit**, предназначенное для автоматизированного сбора (парсинга) уведомлений с публичного портала Единого реестра видов контроля (ЕРВК) — `https://ervk.gov.ru/public/notices`.

## Особенности
Сайт ЕРВК является Single Page Application (SPA), данные которого загружаются динамически через API с использованием сложных одноразовых токенов. 
Вместо попыток реверс-инжиниринга API, в проекте используется **Playwright** — инструмент для автоматизации браузера. Он честно эмулирует действия пользователя:
1. Открывает сайт в скрытом режиме (headless).
2. Разворачивает панель "Расширенный поиск".
3. Вводит заданные пользователем параметры ("Вид деятельности", "Субъект РФ").
4. Ждет загрузки и рендеринга данных.
5. Анализирует структуру DOM-дерева через **BeautifulSoup**, извлекая количество найденных совпадений и примерное число страниц.
6. Собирает данные по каждой карточке уведомления на странице.
7. Автоматически нажимает кнопку перехода на следующую страницу (пагинация) вплоть до последней.
8. Формирует единый `notices.json` файл со всеми найденными записями.

## Стек технологий
* **Python 3**
* **Streamlit** — создание графического веб-интерфейса.
* **Playwright (async)** — автоматизация Chromium-браузера и взаимодействие с веб-элементами.
* **BeautifulSoup4** — парсинг HTML и извлечение необходимых данных (номера уведомлений, адреса, даты и т.д.).

## Структура собираемых данных
Для каждой карточки программа извлекает следующие поля:
* `номер уведомления`
* `наименование хозяйствующего субъекта`
* `дата`
* `наименование уведомления`
* `наименование работы или услуги`
* `коды ОКВЭД`
* `Субъект РФ`
* `Адрес места осуществления деятельности`
* `Наименование контрольного органа`

## Установка и настройка

1. Убедитесь, что у вас установлен Python (рекомендуется версия 3.8+).
2. Перейдите в папку с проектом:
   ```bash
   cd /Users/edwfa/.gemini/antigravity/scratch/scraper
   ```
3. Создайте и активируйте виртуальное окружение (опционально, но рекомендуется):
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
4. Установите зависимости:
   ```bash
   pip install streamlit playwright beautifulsoup4
   ```
5. Установите необходимые бинарные файлы браузеров для Playwright:
   ```bash
   playwright install chromium
   ```

## Запуск приложения

Для запуска интерфейса выполните команду:
```bash
streamlit run app.py
```
После этого в вашем браузере по умолчанию автоматически откроется вкладка с приложением (обычно `http://localhost:8501`).

## Использование
1. При необходимости скорректируйте параметры поиска в полях ввода на веб-странице.
2. Нажмите синюю кнопку **«Начать выгрузку»**.
3. В интерфейсе будет отображаться статус загрузки, количество найденных совпадений и прогресс обхода страниц.
4. По завершении обхода программа выведет первые несколько записей для предпросмотра.
5. Нажмите кнопку **«Скачать notices.json»**, чтобы сохранить все собранные данные к себе на компьютер, либо найдите файл `notices.json` в корневой директории проекта.
